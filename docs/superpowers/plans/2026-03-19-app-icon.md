> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the hand-drawn "A" tray icon with the Claude sparkle logo (white on coloured circle), add a static SVG icon, and a .desktop entry.

**Architecture:** A pre-rendered 40x40 white Claude sparkle PNG is base64-encoded and embedded in tray.py. `make_image()` composites it onto a coloured circle at runtime. A static SVG icon and .desktop file are added for app launcher integration. The .deb build, Makefile, and install.sh are updated to install these new files.

**Tech Stack:** Pillow, SVG, base64, freedesktop .desktop spec

def _load_sparkle() -> Image.Image:
    buf = BytesIO(base64.b64decode(SPARKLE_B64))
    img = Image.open(buf)
    img.load()
    return img.convert("RGBA")

_sparkle = _load_sparkle()


def make_image(color_hex: str) -> Image.Image:
    size = 64
    img = Image.new("RGBA", (size, size), (0, 0, 0, 0))
    draw = ImageDraw.Draw(img)
    draw.ellipse([1, 1, size - 2, size - 2], fill=_hex_to_rgba(color_hex))
    offset = (size - _sparkle.width) // 2
    img.paste(_sparkle, (offset, offset), _sparkle)
    return img
